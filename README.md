## Chat Application

Chat Application

## Installation
```bash
npm i
```

## To run the server
```bash
npm start 
```

## To run the client
```bash
npm start 
```
 
## Description:

1. server -
    1.  Uses Active MQ and Socket.io to send private messages to other users.
    2.  Used JWT for User Validation.
    
2. client - Basic React App with login cred's.
